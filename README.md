# ResearchMind AI — Backend

Enterprise Research Paper Intelligence Platform. RAG system with hybrid retrieval (dense + BM25 + RRF fusion), cross-encoder reranking, and Gemini-powered Q&A, summarization, methodology extraction, paper comparison, and research gap analysis.

Built with FastAPI, FAISS, Sentence-Transformers, and Google Gemini.
